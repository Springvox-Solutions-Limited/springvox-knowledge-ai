# SpringVox User Guide

SpringVox Knowledge AI lets team members ask questions from approved company documents.

## Ask Questions

1. Open `/dashboard/chat`.
2. Ask a question in plain English.
3. Review the answer, confidence badge, sources, and follow-up suggestions.

If SpringVox cannot find support in uploaded documents, it should say so instead of guessing.

## Notifications

Open `/dashboard/notifications` to review platform or workspace notices. You can mark individual notices as read or mark all as read.

## Limits During Beta

Beta environments have rate limits to protect reliability. If you see a rate-limit message, wait for the retry window and try again.
